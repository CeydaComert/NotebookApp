# NotebookApp


 Nesne Tabanlı Programlama dersi kapsamında, C# programlama dili kullanılarak bir proje ödevi hazırlanacaktır. Bu proje ile Visual Studio ortamında bir Windows Form uygulaması geliştirilecek olup, kullanıcı arayüzü tasarımı ve ilgili kodlama süreçleri gerçekleştirilecektir.   




 Within the scope of the Object-Oriented Programming course, a project assignment will be developed using the C# programming language. As part of this project, a Windows Forms application will be created in the Visual Studio environment, including both user interface design and the implementation of relevant coding processes.
